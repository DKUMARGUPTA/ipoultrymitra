## User Authentication & Role Management

The application supports secure Internet Identity authentication with proper routing:
- **Owner**: Highest privilege level with access to all modules, user management, plan control, system-wide settings, and role-based dashboard previews
- **Admin**: Full access to all operational modules and settings
- **Accountant**: Access to accounting, reports, and financial data
- **Supervisor**: Access to poultry management and operational data
- **Dealer**: Access to dealer-specific modules including invoicing and messaging
- **Integrator**: Access to integrator dashboard, farmer management, batch monitoring, and profit sharing calculations
- **Farmer**: Access to farming operations, dealer interactions, and daily input recording

Authentication flow includes:
- Internet Identity integration for secure login
- Automatic redirection to appropriate dashboard based on user role
- Session management and security
- Role-based access control

## Profit Sharing Module

Comprehensive per-batch accounting and settlement system:

### Input Cost Tracking
- Record all supplied inputs per batch:
  - चूजे की लागत / Chick cost - cost per chick and total quantity
  - चारा लागत / Feed cost - total feed supplied and cost
  - दवाई लागत / Medicine cost - vaccination and treatment costs
  - अग्रिम राशि / Advance amount - any advance payments to farmer
- Automatic cost accumulation throughout batch cycle

### Settlement Calculations
- Ready-bird weight recording at batch completion
- Rate per kg configuration for settlement
- Automatic calculation of:
  - कुल आय / Total revenue - ready weight × rate per kg
  - कुल लागत / Total cost - sum of all input costs
  - किसान का हिस्सा / Farmer's share - calculated based on agreement percentage
  - डीलर का मार्जिन / Dealer's margin - remaining profit after farmer's share
- Settlement report generation with detailed breakdown

### Profit Analysis
- Batch profitability analysis with cost breakdown
- Profit margin calculations and trends
- Comparative analysis between different batches
- ROI calculations for integrator and farmer
- Financial performance metrics and reporting
