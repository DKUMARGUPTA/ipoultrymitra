# Contributing to PoultryMitra

Thank you for your interest in contributing to PoultryMitra! We welcome contributions from the community to make this platform better.

## Code of Conduct

- Be respectful and inclusive
- Focus on agricultural domain knowledge
- Maintain code quality standards
- Help each other learn

## How to Contribute

### Reporting Issues
1. Check existing issues first
2. Provide detailed description
3. Include steps to reproduce
4. Add relevant screenshots/logs

### Submitting PRs
1. Fork the repository
2. Create feature branch: `git checkout -b feature/feature-name`
3. Follow coding standards
4. Write tests for new features
5. Update documentation
6. Submit PR with clear description

## Development Setup

```bash
git clone https://github.com/DKUMARGUPTA/ipoultrymitra.git
cd ipoultrymitra
npm install
npm run dev
```

## Testing

```bash
npm run test
npm run lint
```

## Commit Messages

Use conventional commits:
- `feat:` New feature
- `fix:` Bug fix
- `docs:` Documentation
- `style:` Formatting
- `refactor:` Code refactor
- `test:` Tests
- `chore:` Maintenance
